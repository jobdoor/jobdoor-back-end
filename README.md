## Description

This is the main service of JobDoor, where the open positions are managed

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# Development
$ yarn run start:dev

# Stage
$ yarn run start:stage

# Production
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
